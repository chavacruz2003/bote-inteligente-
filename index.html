Guía de Construcción

Bote Inteligente R.E.CO-Z

Gestión automatizada de residuos con Hardware Libre

Descripción del Proyecto

El Bote Inteligente R.E.CO-Z es un sistema automatizado diseñado para identificar y clasificar residuos orgánicos e inorgánicos mediante sensores electrónicos, actuadores y un microcontrolador Arduino Uno R3. El proyecto utiliza herramientas de software libre y componentes accesibles para promover el reciclaje tecnológico.

Fase 1: Preparación y Alistamiento

  • Verificar el correcto funcionamiento del Arduino Uno R3 mediante un test básico (Blink).
  • Revisar minuciosamente que los sensores HC-SR04 (ultrasónico) y YL-69 (humedad) estén limpios y sin obstrucciones.
  • Comprobar continuidad en todos los cables jumper para descartar daños internos.
  • Formatear la tarjeta MicroSD en formato FAT32 (crucial para el módulo DFPlayer).
  • Guardar los archivos de audio en la raíz de la SD dentro de una carpeta llamada mp3, con nombres del 0001.mp3 al 0005.mp3.

Fase 2: Ensamblaje del Circuito

  • Uso de una protoboard de 400 puntos para organizar el montaje central.
  • Conexión del sensor ultrasónico HC-SR04: Pin Trig al digital 6 y pin Echo al digital 7.
  • Conexión del sensor de humedad YL-69: Salida analógica al pin A0.
  • Conexión del módulo DFPlayer Mini: Pines RX/TX a los digitales 2 y 8 (con resistencia de 1kΩ).
  • Conexión del servomotor SG90: Cable de señal al pin digital 4.

Fase 3: Lógica de Programación

  • Desarrollo del código en C++ utilizando el entorno oficial Arduino IDE.
  • Definición de un estado de "reposo" inicial con la tapa del bote cerrada.
  • Activación del ciclo principal al detectar objetos a una distancia inferior a 15 cm.
  • Algoritmo de promedio: Múltiples lecturas de humedad durante 10 segundos para estabilidad.

Fase 4: Integración Mecánica

  • Fijación del servomotor en la parte trasera para accionar el mecanismo de clasificación.
  • Posicionamiento del sensor de humedad YL-69 sobre la plataforma de detección.
  • Protección del Arduino y protoboard dentro de una caja de cartón aislada.

Fase 5: Calibración y Encendido

  • Lectura inicial de humedad ambiental para establecer el punto de referencia.
  • Configuración de ángulos: 10° para orgánico y 160° para inorgánico.
  • Alimentación externa de 5V estable para garantizar torque y calidad de audio.
